Camera Access Error

How to fix camera access:
  1. Check browser address bar for camera permission icon (camera or lock icon)
  2. Click the icon and select "Allow" for camera access
  3. Make sure no other app is using your camera
  4. Try refreshing the page and clicking "Scan QR Code" again
  5. If on mobile, ensure you're using a modern browser (Chrome, Firefox, Safari)

Alternative: You can use the manual input field above to enter reference numbers directly without using the camera.
